# About

我主要感兴趣的方向包括：

- 数论与代数数论；
- 表示论，尤其是有限群表示与 character theory；
- 代数几何，尤其是 schemes、sheaves 与 cohomological methods；
- 交换代数及其在代数几何中的应用。

这个网站主要用于存放：

1. 数学笔记；
2. 读书记录；
3. 学习路线与随笔；
4. 二次元与杂谈。

如果这里的某些内容看起来过于口语化，那大概是因为它们本来就是从学习过程中的草稿慢慢整理出来的。
